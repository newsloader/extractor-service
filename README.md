# extractor-service

# Setup

```sh
git clone git@github.com:newsloader/extractor-service.git
cd extractor-service
cp example.env .env
pnpm i
pnpm dev
```

Implemented endpoints:

- http://0.0.0.0:20101/
- http://0.0.0.0:20101/api/health
- http://0.0.0.0:20101/api/sicom/article?url={SI.COM_LINK}
